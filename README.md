# DAO Governance Pilot Analysis

Preliminary analysis of governance concentration across Ethereum DAOs using on-chain voting data from the Snapshot GraphQL API.

## Overview

This project computes governance concentration metrics (Gini coefficient, top-10 voter share) for six major Ethereum DAOs: ENS, Gitcoin, Uniswap, Arbitrum, Optimism, and Lido. Institutional design conditions are coded and compared against governance outcomes.

## Data Sources

- Snapshot GraphQL API (voting data)
- Governance documentation (institutional conditions)

## Structure

- scripts/ — Python scripts for data collection and analysis
- data/ — Raw and summary governance data
- outputs/ — Charts and visualisations
